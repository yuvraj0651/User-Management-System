# 👥 User Management System 

## 📌 Overview
A scalable User Management System built to handle user data, roles, and access control in a structured and efficient way. This application simulates real-world admin panel workflows, enabling administrators to manage users with full CRUD operations and role-based access.

## 🛠️ Tech Stack
- React.js
- Redux Toolkit
- Tailwind CSS
- JavaScript (ES6+)
- REST APIs / JSON Server

---

## ✨ Features
- 👥 User management (Create, Read, Update, Delete)
- 🔐 Role-based access control (Admin / User)
- 🔍 Search, filter, and sorting functionality
- 📋 Dynamic data table with reusable components
- 🔄 Global state management using Redux Toolkit
- ⚙️ Form validation for user inputs
- 🔒 Protected routes based on user roles
- 📱 Fully responsive admin interface
- 🌙 Dark mode support

- Building reusable and scalable admin UI components
