# 👥 User Management System

## 📌 Overview
A scalable user management system that allows admin users to manage users, roles, and permissions with secure authentication and role-based access control.

## 🛠️ Tech Stack
- React.js
- Redux Toolkit
- Tailwind CSS

## ✨ Features
- 🔐 Authentication system
- 👤 User CRUD operations
- 🛡️ Role-based access control
- ⚙️ Admin dashboard interface

## 🧠 Key Learnings
- Role-based routing
- Managing complex state
- Building scalable admin systems

## 🚀 Future Improvements
- Backend integration
- Activity logs
- Advanced permissions system
